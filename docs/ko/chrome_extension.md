# Onyx Chrome 확장 프로그램

Onyx Chrome 확장 프로그램은 팀의 고유한 지식으로 구동되는 LLM을 통해 브라우저에서 직접 연구, 생성, 자동화를 가능하게 합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **빠른 접근** | Mac은 `Ctrl + O`, Windows는 `Alt + O`로 즉시 Onyx 사이드바 열기 |
| **채팅 인터페이스** | 기능과 상호작용하는 자연어 채팅 인터페이스 |
| **내부 검색** | 팀의 50개 이상 커넥터 기반 모든 도구에서 답변 검색 |
| **요약** | 모든 작업 앱의 정보를 즉시 요약 |
| **새 탭 페이지** | 회사의 모든 지식에 빠르게 접근 |
| **AI 에이전트** | 고유한 사용 사례를 위한 커스텀 AI 에이전트 접근 |

---

## 연동 가능한 앱

Onyx는 Google Drive, Jira, Confluence, Slack 등 수십 개의 인기 직장 앱과 연동됩니다.

> **참고:** 이 확장 프로그램은 팀 관리자가 계정을 생성한 경우 사용하세요.

---

## 설치

**Onyx Cloud 사용자**: Chrome 플러그인 스토어 방문 (승인 대기 중)
